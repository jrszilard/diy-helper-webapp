Key shared elements across all options

All layouts include

  • Two tabs only: "Ask Anything" (unified AI) + "Ask an Expert"
  • Full chat features (projects, materials, shopping list, tools inventory)
  • Value prop differentiators visible to new users
  • Quick-start suggestion chips
  • Dark theme (current landing page aesthetic)

Trade-offs to consider

  • A: Simple but value prop may get scrolled past quickly
  • B: Most feature-rich view but complex on mobile
  • C: Best first impression but needs animation work for the morph
